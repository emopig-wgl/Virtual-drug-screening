# Virtual Drug Screening Based on Transcriptomic Analysis, Topological Network Analysis and Machine Learning

## Overview

This repository provides the complete computational framework developed in our study for identifying potential therapeutic compounds against Alzheimer's disease through the integration of transcriptomic analysis, topological network analysis, and machine learning-based virtual screening.

The workflow consists of four major stages:

1. Identification of differentially expressed genes (DEGs) from publicly available transcriptomic datasets.
2. Screening of disease-associated key genes using Persistent Homology (PH) and Persistent Spectral Theory (PST).
3. Construction of machine learning models for molecular binding affinity prediction.
4. Large-scale virtual screening of DrugBank compounds.

# 1. Differential Expression Analysis

Seven Alzheimer's disease transcriptomic datasets were downloaded from the Gene Expression Omnibus (GEO) database.

For each dataset:

- Differentially expressed genes (DEGs) were identified.
- Protein–protein interaction (PPI) networks were constructed.
- Network topology was analyzed under multiple interaction confidence thresholds.

The resulting PPI networks served as the input for subsequent topological analyses.

---

# 2. Persistent Homology and Persistent Spectral Theory

## Persistent Homology (PH)

Persistent Homology characterizes the evolution of connected components and higher-order topological structures throughout the filtration process.

For each node, the corresponding topological changes after node removal were quantified to evaluate its structural importance.

## Persistent Spectral Theory (PST)

Persistent Spectral Theory measures changes in the eigenvalue spectrum of graph Laplacians during filtration.

Unlike traditional centrality measures, PST captures multiscale structural perturbations induced by node deletion.

## Key Gene Identification

The rankings generated by PH and PST under multiple filtration thresholds were integrated.

Genes consistently ranked highly across different datasets and filtration scales were retained as key genes.

After removing duplicated genes among all datasets, a final set of 67 key genes was obtained.

---

# 3. Machine Learning Pipeline

## 3.1 ChEMBL Dataset Construction

The identified key genes were mapped to the ChEMBL database.

Only targets satisfying the following criteria were retained:

- Human protein targets
- Experimentally validated bioactivity records
- Sufficient sample size for model training

Finally,

- 5 target-specific ChEMBL datasets
- 4,492 bioactivity samples

were selected for downstream machine learning.

---

## 3.2 Molecular Feature Extraction

Two molecular representations were employed.

### Autoencoder (AE)

Continuous Data-Driven Descriptors (CDDD) were generated using a pretrained molecular autoencoder.

Each SMILES string was encoded into a continuous 512-dimensional latent vector.

### Extended Connectivity Fingerprints (ECFP)

ECFP fingerprints were generated using the Morgan algorithm (radius = 2).

These fingerprints encode local molecular substructures into fixed-length binary vectors.

### Feature Fusion

The final molecular representation was obtained by concatenating the AE and ECFP descriptors.

```text
AE feature
        │
        ▼
ECFP feature
        │
        ▼
Feature concatenation
        │
        ▼
AE + ECFP representation
```

---

## 3.3 Machine Learning Models

Three regression algorithms were evaluated:

- Support Vector Regression (SVR)
- Random Forest (RF)
- Gradient Boosting Decision Trees (GBDT)

Each model was trained using

- AE descriptors
- ECFP descriptors
- AE + ECFP fused descriptors

Performance was evaluated using

- Pearson Correlation Coefficient (PCC)
- Coefficient of Determination (R²)
- Root Mean Square Deviation (RMSD)

Among all tested combinations, GBDT with AE + ECFP achieved the best predictive performance and was selected for virtual screening.

---

# 4. Virtual Screening

The optimized GBDT model was applied to the DrugBank database.

For each molecule:

- AE descriptors were generated.
- ECFP fingerprints were computed.
- Fused molecular representations were constructed.
- Binding affinities toward five therapeutic targets were predicted.

Candidate compounds were ranked according to their predicted binding affinities for downstream biological validation.
